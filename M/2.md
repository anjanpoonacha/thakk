# मंगलत्र अडिगे माने'ल ओंध थोंधरे
## Mangalathra adige mane'l ondh thondhare
### कोडव तक्कमधली एक गोष्ट — मराठी अनुवादासह

---

> कोडव मूळ वक्त्याकडून मान्यता मिळालेली कथा.
> सर्व दुरुस्त्या kodava_corrections.md मध्ये नोंदवलेल्या आहेत.

---

## भाग एक: बेंगळूरु

किरण त्याचं नाव. कोडगूच्या डोंगरांमधून आलेला, बेंगळूरुच्या गर्दीत हरवलेला.
दोन वर्षे झाली — ऑफिस, ट्रॅफिक, आणि एकटेपणा.

naan Kiran. Naa Kodagu'nja bandhiye. Bengaluru'l ullo.
*(माझं नाव किरण. मी कोडगूहून आलो. बेंगळूरुत आहे.)*

आज मनात एक जड भावना होती. त्याने फोन उचलला.

naa'k ikka dhand kaala Bengaluru'l kelsath'l ullo. naa indh baarii beajarat'l uLL.
*(दोन वर्षांपासून बेंगळूरुतल्या कामात आहे. आज खूप जड वाटतंय.)*

---

## भाग दोन: ताथाचा आवाज

फोन वाजला. एकाच रिंगमध्ये उत्तर आलं — जणू ताथा वाट पाहत होते.

Thaatha: **en visheasha, Kiran?**
*(काय बातमी, किरण?)*

किरणच्या घशात काहीतरी अडकलं. शब्द हळूहळू बाहेर पडले.

Kiran: **visheasha ondh ille, thaatha. naa dhumba beajarat'l uLL.**
*(काही बातमी नाही, ताथा. खूप जड वाटतंय.)*

बेंगळूरुतली रात्र डोळ्यांसमोर तरळली — हजारो माणसे, हजारो गाड्या, पण आपल्यासाठी कोणीच नाही. सगळ्यात जास्त आठवण येत होती ती त्या माने ची — त्या अवया ची, त्या ताथांची.

**Bengaluru'l sakkath jana injathe, sakkath car injathe — aachenge naada mane'na, naada thaatha'na, naada awaya'na naa'k dhumba ikka ghê-na aapa.**
*(बेंगळूरुत खूप माणसे, खूप गाड्या — पण आमच्या घराची, ताथांची, अवयाची खूप आठवण येते.)*

ताथा हसले. त्या हसण्यात माया होती, चिडवणूकही होती.

Thaatha: **[thoLchathe] ai Kiran!**
*(हसत: अरे किरण!)*

**nii Bengaluru'k poachenge, nanga êNNiiye — "thaatha, naa baari chaayi'l ippii!" adh'na nii marandhith'le?**
*(बेंगळूरुला जाताना तू म्हणालास — "ताथा, मी खूप छान राहीन!" — ते विसरलास का?)*

किरण हसला — जरासा लाजत, जरासा ओशाळत.

Kiran: **akku thaatha, naa marandhith'le. aachenge illi oar chuur serii aap'le.**
*(हो ताथा, विसरलो नाही. पण इथे एक चूरही ठीक वाटत नाही.)*

---

## भाग तीन: खूशखबर

मग ताथांनी ती गोष्ट सांगितली जी किरणला उचलून उभं करणार होती.

Thaatha: **Kiran! bappa thingath'l Sindhu akkanda mangala injathe!**
*(किरण! पुढच्या महिन्यात सिंधू अक्कांचं लग्न आहे!)*

**nii bakkondu, baaLe kaara!**
*(तुला यायलाच हवं, तरुण माणसा!)*

किरण एकदम उसळला — जणू बेंगळूरुचं ते सारं ओझं एका क्षणात गळून पडलं.

Kiran: **[thoLchathe] akku thaatha! naa bappii!**
*(हसत: हो ताथा! मी येतो!)*

---

## भाग चार: घरी परतणे

दोन आठवडे उलटले.

**Sumaar dhand waara poayith. Kiran mane'k baathe.**
*(सुमारे दोन आठवडे गेले. किरण घरी आला.)*

अडिगे माने च्या दिशेने एक खास घमघमाट येत होता. घरातला तो परिचित वास — नेयी आणि मसाल्यांचा — नाकात शिरला आणि काहीतरी आतून सैल झालं.

**Awaya adige mane'l koaLi karii maadiyand injathe.**
*(अवया स्वयंपाकघरात कोळी कारी बनवत होत्या.)*

**neyi'ra ghama injathe.** [⚠️ ghama — native speaker supplied]
*(तुपाचा घमघमाट होता.)*

अवयाने दरवाजाकडे नजर टाकली — कोण आलंय? — आणि मग —

**Awaya bandê jana'la noot'chi — "dhaar idh?..." — KIRAN!!!**
*(आलेल्या माणसांकडे पाहिलं — "कोण हे?..." — किरण!!!)*

किरणने वाकून अवयाचे पाय शिवले — त्या एका स्पर्शात दोन वर्षांचा थकवा होता.

**Kiran kaal pudichiyand — "awaya!" — ava'k kaNN'ella niir banthê.**
*(किरणने पाय स्पर्श केले — "अवया!" — त्यांच्या डोळ्यांत पाणी आलं.)*

---

## भाग पाच: लग्नाचा दिवस — मुख्य प्रसंग

**Mangalathra dhivsa baathe.**
*(लग्नाचा दिवस आला.)*

मंगळथ्र च्या गोंधळात सगळे गुंतलेले होते. किरण अडिगे मानेत शिरला — तिथे पाथ्रे चमकत होते, वाफा निघत होत्या.

**Kiran adige mane'k poayith.**
*(किरण स्वयंपाकघरात गेला.)*

एक मोठं, बिसी भांडं समोर होतं —

**Ava'k oar bisi paathre kaNdathe — chuduva'cchak bisi injathe.** [⚠️ chuduva'cchak — native speaker supplied]
*(त्याला एक गरम भांडं दिसलं — जळजळीत गरम होतं.)*

आणि मग किरणच्या मनात एक निर्दोष विचार आला — "हे मला हवं आहे."

**"naa'k adh boaNdu" êNNiyand ava aa paathre'na êdp'k suruu maadiye —**
*("मला ते हवं आहे" म्हणत तो ते भांडं उचलायला लागला —)*

**POTTI POCHI!!!**
*(फुटलं आणि पडलं!!!)*

**Bisi kuuL'ra, koaLi karii'ra, beembaLe karii'ra — mane thira!!!**
*(गरम भात, कोंबडी रस्सा, बांबूची भाजी — घरभर!!!)*

एक क्षण शांतता. मग —

**Bojavva [बोजव्वा = आत्या]: KIRAAAAN!!!**

किरण हळूच मागे सरकला, हात वर करत —

**Kiran: wondh aap'le... wondh aap'le...**
*(किरण: काही झालं नाही... काही झालं नाही...)*

पण ते भांडं, तो आवाज, तो गोंधळ —

**Mangala mane'l ellaru thoLcha't.**
*(लग्नाच्या घरातले सगळे हसले.)*

**Thaatha'saa thoLchathe. Bojavva'saa thoLchathe.**
*(ताथा पण हसले. बोजव्वा पण हसल्या.)*

---

## भाग सहा: फिल्टर काफी आणि शेवट

दिवस उगवला. कोडगूच्या थंड हवेत filter kaapi चा सुगंध पसरला होता.

**Oar dhivsa — polaaka'l filter kaapi kudchiyand aap'ka — Thaatha êNNiiye:**
*(एक दिवस — सकाळी फिल्टर काफी पित असताना — ताथा म्हणाले:)*

ताथांनी किरणकडे पाहिलं. एक मिश्कील हास्य त्यांच्या चेहऱ्यावर होतं.

**Thaatha: Kiran, nii minynya mangala mane'l aa paathre pott'chittiya'la —**
*(किरण, तू आधी लग्नाच्या घरात ते भांडं फोडलेस —)*

**aan'g naada'ella adige ella'k inyii rasa aachi, naa'k aapa!**
*(त्यामुळे आमचं सगळं जेवण आणखी रुचकर झालं, असं मला वाटतं!)*

**Kiran: THAATHA!!!**

**Thaatha thoLchathe.**
*(ताथा हसले.)*

**Mane'l ellaru thoLcha't.**
*(घरातले सगळे हसले.)*

**Kodag'ra gaaLi thoLchathe.**
*(कोडगूचा वारा हसला.)*

| awaya | आजी | **अ-व-य** — अ = "अंबर"मधला ऱ्हस्व अ, व = "वाट"मधला व, य = "याद"मधला य | |
| mane | घर | **म-ने** — म = "मन"मधला म, ऱ्हस्व अ ("मन"सारखा), ने = "नेट"मधला ने | |
| adige mane | स्वयंपाकघर | **अ-डि-गे म-ने** — ड = मागे वळलेल्या जिभेने (मूर्धन्य ड), गे = "गेट"मधला गे | ड = मूर्धन्य |
| koaLi | कोंबडी | **को-ळि** — को = "कोण"मधला को (दीर्घ), ळि = मराठी ळ + इ | ळ = मूर्धन्य |
| karii | आमटी / रस्सा | **का-री** — का = दीर्घ आ ("काम"), री = दीर्घ ई ("रीत") | दीर्घ आ आणि ई |
| neyi | तूप | **ने-यि** — ने = "नेट"मधला ऱ्हस्व ए, यि = "यित्"सारखा य + ऱ्हस्व इ | |
| ghama | सुगंध / घमघमाट | **घ-मा** — घ = मराठी घ, मा = "मन"मधला ऱ्हस्व म + अ | ⚠️ मूळ वक्त्याने दिलेला |
| paathre | भांडे | **पा-थ्रे** — पा = दीर्घ आ, थ्रे = "ताट"मधला त + ऱ्हस्व रे — महाप्राण थ नाही | थ = दंत्य |
| bisi | गरम | **बि-सि** — बि = ऱ्हस्व इ ("बित्ती"), सि = ऱ्हस्व इ ("सित") | |
| mangala | लग्न | **मं-ग-ल** — मं = "मंगळ"मधला मं, ग = "गम"मधला ग, ल = "लव"मधला ल | |
| beajarat | दुःख / जड मन | **बे-अ-ज-रत** — बे = दीर्घ ए ("बेत"), अ = ऱ्हस्व, ज = "जड"मधला ज, र = मृदु र, त = दंत्य त | |
| thoLchathe | हसले (भूतकाळ) | **थो-ळ-च-थे** — थो = दंत्य थ + ओ, ळ = मराठी ळ, च = "चल"मधला च, थे = दंत्य थ + ए | थ = दंत्य (महाप्राण थ नाही); ळ = मूर्धन्य |
| ellaru | सगळे | **एल्-ल-रु** — एल्ल = द्विगुणित ल ("एल् ल" जोडून उच्चारा — दुप्पट वेळ धरा), रु = "रुप"मधला रु | ल्ल = द्विगुणित |

| chuduva'cchak | जळजळीत गरम | **चु-दु-व'च्-चक** — चु = "चुल"मधला च + उ, दु = दंत्य द + उ, च्च = द्विगुणित च (दुप्पट) | ⚠️ मूळ वक्त्याने दिलेला; च्च = द्विगुणित |

| Kodag'ra | कोडगूचा/ची | **को-डग्-र** — को = दीर्घ ओ, ड = मूर्धन्य ड, ग् = जोडलेला ग, र = मृदु र | षष्ठी विभक्ती प्रत्यय -'ra |

| baaLe kaara | तरुण माणूस | **बा-ळे का-र** — बा = दीर्घ आ, ळे = मराठी ळ + ए, का = दीर्घ आ, र = मृदु र | ळ = मूर्धन्य |
