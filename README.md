# Thakk — Kodava Takk Language Resource

A language resource for Kodava takk — a Dravidian language spoken in Coorg (Kodagu), Karnataka. This repo is the single source of truth for the [kodava-rag](https://github.com/anjanpoonacha/kodava-rag) system, which answers questions about Kodava grammar, vocabulary, and sentence construction.

## Data Sources and Truth Hierarchy

Two-tier truth system:

1. **`elementary_kodava_FINAL.md`** — base knowledge layer. A complete Kodava takk textbook covering phonology, grammar, vocabulary, and sentence structure.

2. **`kodava_corrections.md`** — native speaker override layer. Corrections from a native Kodava speaker that supersede any conflicting information in the textbook. Entries here always take precedence.

The override layer exists because published textbooks occasionally contain romanization inconsistencies or outdated usages.

## Flag System

| Flag | Meaning |
|------|---------|
| ✅ | High confidence — verified by a native speaker |
| ⚠️ | Uncertain — no clean equivalent; treat with caution |
| 🔴 | Grammar trap — structural rule commonly misapplied (e.g. geminate consonants, retroflex distinctions) |
| 🟡 | Provisional — sourced from textbook only; not yet native-speaker verified |

## Adding New Corrections

1. Append a block to `kodava_corrections.md` using the `WHAT / CORRECT / WHY / CONFIDENCE` format.
2. Run `make corpus` in `kodava-rag` — the RAG index updates immediately.

See `corpus/README.md` for the full contributor workflow.
