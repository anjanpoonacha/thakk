## Days of the Week — Aachekal

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Sunday | naaraache | naara = sun; aache = day |
| Monday | thingLaache | thingLa = moon; aache = day |
| Tuesday | chowaache | chowa = Mars; aache = day |
| Wednesday | padhnaache | padhna = Mercury; aache = day |
| Thursday | beaLaache | beaLa = Jupiter; aache = day |
| Friday | boLiyaache | boLi = Venus; aache = day |
| Saturday | cheniaache | cheni = Saturn; aache = day |
