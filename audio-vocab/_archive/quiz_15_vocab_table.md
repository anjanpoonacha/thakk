# Quiz 15 — Kodava Padipu Vocab Table

**Topic:** Kodava culture — banana varieties, tools, ceremonies, farming implements, measuring instruments, festivals, furniture, chilli types, rivers, wedding attire

## Banana Types Found in Kodagu — Kodag'l buLeva baaLe

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Flower banana | puu baaLe | puu = flower; baaLe = banana |
| Forest/tree banana | mara baaLe | mara = tree/forest; baaLe = banana |
| Black/dark banana | kari baaLe | kari = black/dark; baaLe = banana |

| English | Kodava Takk | Explanation |
