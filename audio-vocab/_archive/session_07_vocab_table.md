## Kailpodh Festival

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| We Kodavas have three festivals. | Nangak Kodava'k muundh namme uNd | |
| The three main Kodava festivals | Kailpodh, Kaveri Sankramana, Puthari | |
| Today we will talk about Kailpodh. | Indh nanga Kailpodh'ra vishya thakk pariyana | |
| Kailpodh is celebrated after paddy transplantation work is done. | Kailpodh namme naati paNi aana pinynya celebrate maaduva | naati paNi = transplanting work |
| Kailpodh is celebrated on September 3rd. | Kailpodh nammena September muundhangu celebrate maaduva | |
| The Kodava month of Cheeniyaar | Kodava thinga Cheeniyaar | thinga = month |
| On that day we worship weapons (guns and swords). | Andh nanga thooku pinya kaththirantha aayudhathana pooje maaduva | kaththirantha = sword; aayudhathana = weapons; thooku = gun |
| We have a grand feast after the worship. | Nanga pooje aana pinya gaddaLa umba | |
| Tasty/feast food | gaddaLa | |
| Kadambuttu and pork curry are the special Kailpodh dishes. | Kailpodh'k Kadambuttu pinya pandi kari gaddaLa | |
| Ancestors | guru kaarananga | |
| Alcohol is offered to the ancestors. | Guru kaaranangak kaLLu meedhi beppa | kaLLu = rice beer/alcohol; meedhi = offering |
| Offer / to place an offering | meedhi | |
| Competition | paipoti | |
| Coconut shooting and vaalaga dance competitions are held. | Thenge bodi pinya vaalaga thaattu intha paipoti nadatthuva | thenge bodi = coconut shooting; vaalaga = traditional dance |
| Family / Clan | wokka | Refers to the entire clan (e.g., PaLiyanda, Bopanda, Monnanda) |
| The whole clan celebrates the festival together. | Wokka ella koodithth wakkache nammena nadatthuva | |

## Household Commands

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Come here. | Ilik ba | ilik = here; ba = come (imperative) |
| Let's go there. | Alik pokana | alik = there; pokana = let's go |
| Don't go there. | Alik pokathe | -athe = negative imperative |
| Give it to me / Give me. | Naak'thaa | naa'k = to me; thaa = give |
| Give it to her. | Ava'k kodu | ava'k = to her; kodu = give |
| Give it to him. | Avang'kodu | avang'k = to him |
| Wash your hands. | Kai kathth | kai = hand; kathth = wash |
| Go wash your hands and come. | Po kai kaththith'ba | po = go; ba = come |
| Come to eat. | Umba'k ba | umba = food/eat; 'k ba = come to |
| Let us talk. | Nanga thakk pariyana | thakk = speech/language; pariyana = let us speak |
| Don't talk. | Thakk pariyatte | -atte = negative imperative |
| Do your homework. | Homework maad | maad = do (imperative of maaduw'k) |
