# Quiz 18 — Kodava Padipu Vocab Table

**Topic:** Kannada words replaced by authentic Kodava Takk equivalents — vocabulary reclamation

## Kodava Takk Words — inda Kodava pada?

*Format: Kannada/common word → correct Kodava Takk word*

| Common Word (Kannada) | Kodava Takk | Explanation |
|----------------------|-------------|-------------|
| Shunti (dry ginger) | inji | dry ginger; inji is the Kodava form |
| Kaalchainu / kaalgejje (anklet) | kaachaara | traditional Kodava ankle ornament |
| Muukutti (nose ring) | muu kodupu | muu = nose; kodupu = nose ornament |
| Buruli (heel) | berchappa | back/heel of the foot |
| Chenni dandu / chain | chinnitaaru / taar kali | chain or chained bangle |
| Sonta (half) | are | half; are is the Kodava word for half |
| Vayassu (age) | bayath | age in Kodava |
| Mane (house — Kannada form) | aarooda | the Kodava word for one's own home/household |
| Tondekaayi (ridge gourd) | kaavakke | ridge gourd in Kodava |
| Bhaara (load/weight) | kana | weight/load in Kodava |
