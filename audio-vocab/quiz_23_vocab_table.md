# Quiz 23 — Kodava Vocabulary

> **Source:** Kodava Padipu — Kodava Quest (insects quiz)
> **Format:** English word shown → `idhungu Kodava paajje'l entha'ndh aNNuwaa?` (What do you call this in Kodava?) → Kodava answer

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Firefly | minnam puLu | minnam = flash/lightning; puLu = worm/insect |
| Spider | koaLamber | - |
| Butterfly | paappuli | - |
| Snail | koaTeinji / koaTanji | koaTe = shell |
| Grasshopper | pachche pukka | pachche = green; pukka = insect |
| Scorpion | nyaNdrukka | related to nyaNdu (crab) |
| Wasp | kadundhi | - |
| Cockroach | paathe | - |
| Honeybee | thenu oLanji | thenu = honey; oLanji = bee/fly |
| Food fly (housefly) | muuLi | common housefly |
