# Quiz 27 — Kodava Vocabulary

> **Source:** Kodava Padipu — Kodava Quiz (fruits)
> **Format:** English fruit name → `Kodava paaje'l entha'ndu eNNuva?` (What do you call it in Kodava?) → Kodava answer

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Pineapple | Parangi chekke | parangi = foreign; chekke = jackfruit; lit. "foreign jackfruit" |
| Carambola (Star fruit) | Miinaakshi paNNu | paNNu = fruit |
| Muskmelon | Minuke paNNu | paNNu = fruit |
| Mulberry | Chippli paNNu | paNNu = fruit |
| Jamun (Indian blackberry) | Neerale paNNu | paNNu = fruit; neerale = regional name for jamun |
| Dates | Iithe paNNu | paNNu = fruit |
| Fig | Atthi paNNu | paNNu = fruit; atthi = fig tree (also in Kannada) |
| Watermelon | Baththange | - |
| Jackfruit | Chekke | base word also used in Parangi chekke (pineapple) |
| Pomelo | Chakkotha | - |

## Topics Covered
- Fruits — Kodava names
- Word formation: paNNu (fruit) as a productive suffix for naming fruits
