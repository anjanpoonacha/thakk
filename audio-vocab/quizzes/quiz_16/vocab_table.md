# Quiz 16 — Kodava Vocabulary (Wildlife)
> **Source:** quiz_16.mp3
> **Format:** Wildlife vocabulary quiz — English prompt, Kodava Takk answer

| English | Kodava Takk | Kannada Script | Explanation |
|---------|-------------|----------------|-------------|
| What is this called in Kodava Takk? | Idhange Kodava thakk'l enth'andh aNNuva? | ಇದಂಗೆ ಕೊಡವ ತಕ್ಕಲ್ ಎಂತಂದ್ ಅಣ್ಣುವ? | Standard quiz question phrase: "What is this called in Kodava Takk?" |
| Rabbit | Mona | ಮೊನ | Kodava word for rabbit or hare. |
| Stag | Kadama | ಕಡಮ | Refers to the Sambar deer, large deer native to the Indian subcontinent. |
| Squirrel | Koriya | ಕೊರಿಯ | Kodava word for squirrel. |
| Snail | Kattenji | ಕಾಟೆಞ್ಞಿ | Kodava word for snail. |
| Woodpecker | Marakotta pakshi | ಮರಕೊಟ್ಟ ಪಕ್ಷಿ | Literally "tree-pecking bird"; mara=tree, kotta=pecked, pakshi=bird. |
| Bison | Kaati | ಕಾಟಿ | Refers to the Gaur (Indian bison), native to the Western Ghats. |
| Vulture | Jaddu paththu | ಜಡ್ಡ್ ಪತ್ತ್ | Kodava compound for vulture; paththu=bird. |
| Wild Boar | Kaadu pandi | ಕಾಡ್ ಪಂದಿ | Literally "forest pig"; kaadu=forest/wild, pandi=pig. |
| Leopard | Beange nari | ಬೇಂಗೆ ನರಿ | Literally "spotted fox/jackal"; beange=spotted, nari=fox/jackal. |
| Bulbul | Kottumudicha | ಕೊಟ್ಟುಮುಡಿಚ | Crested songbird; name likely refers to its distinctive crest. |

## Topics Covered
- Wildlife — mammals (rabbit, stag, squirrel, bison, wild boar, leopard)
- Wildlife — birds (woodpecker, vulture, bulbul)
- Wildlife — mollusks (snail)
- Quiz question structure in Kodava Takk
