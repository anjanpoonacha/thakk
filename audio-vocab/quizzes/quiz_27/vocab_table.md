# Quiz 27 — Kodava Vocabulary
> **Source:** quiz_27.mp3
> **Format:** Fruit names quiz — Kodava Takk names for 10 fruits

| English | Kodava Takk | Kannada Script | Explanation |
|---------|-------------|----------------|-------------|
| Pineapple | Parangi chekke | ಪರಂಗಿ ಚಕ್ಕೆ | Kodava name for pineapple; parangi = foreign, chekke = jackfruit-type fruit |
| Carambola (Star fruit) | Meenakshi paNN | ಮೀನಾಕ್ಷಿ ಪಣ್ಣ್ | Kodava name for star fruit; paNN = fruit |
| Muskmelon | Minukke paNN | ಮಿನುಕ್ಕೆ ಪಣ್ಣ್ | Kodava name for muskmelon; minukke = shiny/glittery |
| Mulberry | Chippli paNN | ಚಿಪ್ಲಿ ಪಣ್ಣ್ | Kodava name for mulberry |
| Jamun (Java plum) | Nerale paNN | ನೇರಳೆ ಪಣ್ಣ್ | Kodava name for jamun; nerale = the jamun tree |
| Dates | Iithe paNN | ಈತೆ ಪಣ್ಣ್ | Kodava name for dates; iithe = date palm |
| Fig | Atthi paNN | ಅತ್ತಿ ಪಣ್ಣ್ | Kodava name for fig; atthi = fig tree |
| Watermelon | Batthange | ಬತ್ತಂಗೆ | Kodava name for watermelon |
| Jackfruit | Chekke | ಚಕ್ಕೆ | Kodava name for jackfruit |
| Pomelo | Chakkotha | ಚಕ್ಕೋತ | Kodava name for pomelo |
| What is it called in the language? | paaje'l enth'endh uNNuva | ಪಾಜೆಲ್ ಎಂತಂದ್ ಉಣ್ಣುವ | Quiz question phrase; paaje'l = in the language, enth'endh = what, uNNuva = is it called |
| Kodava Study / Learning | Kodava Padippu | ಕೊಡವ ಪಡಿಪು | Name of the presenting group; padippu = study/learning |
| The team's presentation | Tandathira prastuti | ತಂಡತಿರ ಪ್ರಸ್ತುತಿ | Intro phrase; tanda = team, -thira = of the, prastuti = presentation |

## Topics Covered
- Fruit names (Kodava Takk equivalents for 10 fruits)
- Quiz / question phrases
- Program introduction vocabulary
