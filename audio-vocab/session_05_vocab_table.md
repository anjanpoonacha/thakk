## Numbers (41–50)

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Forty-one (41) | naapath'ondh | Example: pathth guuma (10 owls) + nuppath'ondh guuma (31 owls) = 41 |
| Forty-two (42) | naapath'dhand | Example: irovadh kappe (20 frogs) + irovadh'dhand kappe (22 frogs) = 42 |
| Forty-three (43) | naapath'muundh | Example: nuppadh biiga (30 locks) + padhimuundh biiga (13 locks) = 43 |
| Forty-four (44) | naapath'naal | Example: naapadh kondi (40 monkeys) + naal kondi (4 monkeys) = 44 |
| Forty-five (45) | naapath'anjii | Example: pathth motte (10 eggs) + nuppath'anjii motte (35 eggs) = 45 |
| Forty-six (46) | naapath'aar | Example: irovadh chakra (20 wheels) + irovath'aar chakra (26 wheels) = 46 |
| Forty-seven (47) | naapath'êêL | Example: nuppadh (30 keys) + padhinêêL (17 keys) = 47 |
| Forty-eight (48) | naapath'êtt | Example: naapadh jana (40 people) + êtt jana (8 people) = 48 |
| Forty-nine (49) | naapath'woyimbadh | Example: pathth pakshi (10 birds) + nuppath'woyimbadh pakshi (39 birds) = 49 |
| Fifty (50) | ayimbadh | Example: irovadh bera (20 fingers) + nuppadh bera (30 fingers) = 50 |

## Kinship Terms (Paternal — bendhu chaachen)

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Paternal relationships | bendhu chaachen | bendhu = relatives, chaachen = father's side |
| Grandfather (paternal) | thaatha | Father's father |
| Grandmother (paternal) | avaya | Father's mother |
| Father's older brother | ballyappa | ballya (big/elder) + appa (father-figure) |
| Father's brother (general) | bojappa | Middle paternal uncle |
| Father's younger brother | kunyappa | kuny (small/younger) + appa |
| Father's sister | maami | All of father's sisters; also called aunt |
| Wife of ballyappa | ballyava | ballya (elder) + ava (woman/wife) |
| Wife of bojappa | bojava | |
| Wife of kunyappa | kunyava | kuny (younger) + ava |
| Elder male cousin (paternal) | anna | Father's brother's son, if older than you |
| Elder female cousin (paternal) | akka | Father's brother's daughter, if older than you |
| Husband of maami | maama | Uncle by marriage (father's sister's husband) |
| Father's older sister | ballyamaami | ballya (elder) + maami |
| Father's sister (general) | bojimaami | Also said as bojmaami |

## Sample Sentences

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| How much did it become? | Echchakaachi? | echchak (how much) + aachi (did it become); used after counting |
| When added, how much did it become? | Kuudok'ka echchakaachi? | kuudok'ka = when added together |
| What animal do you see here? | niik' illi eadh praaNi'na kaamba? | niik' (you), illi (here), eadh (which), praaNi (animal), 'na (accusative), kaamba (see) |
| I see a cow here. | naak' illi oru payyu kaamba. | naak' (I), oru (a/one), payyu (cow), kaamba (see) |
| What all do you see here? | ningak' illi enth'ella kaamba? | ningak' (you, formal/plural), enth'ella (what all) |
| I see a banana. | naak' oru baaLe paNNu kaamba. | baaLe paNNu (banana) |
| Here, I see a three-legged stool. | illi naak' oru mukkaali kaamba. | illi (here), mukkaali (three-legged stool) |
