## Numbers 81-90

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| Do you remember what eighty is called? | Gyaan uNdane ninga'k eighty'k entha ênnuw'd? | Question from instructor. |
| 80 | êNbadh | Eighty |
| 81 | êNbath-ondh | Eighty-one |
| 82 | êNbath-dhand | Eighty-two. Example: nuppadh (30) + aimbath-dhand (52) = 82 |
| Example sentence | kêNNa makka nuppadh pinynya aimbath-dhand kêNNa makka kuudun'k echchak aapa? êNbath-dhand. | 30 boys plus 52 boys, how many does it become? 82. |
| 83 | êNbath-muundh | Eighty-three. Example: naapadh (40) + naapath-muundh (43) = 83 |
| 84 | êNbath-naal | Eighty-four. Example: aimbadh (50) gaalipatta + nuppath-naal (34) gaalipatta = 84 gaalipatta (eucalyptus leaves) |
| Deer | miga / jinkya | miga or jinkya can be used for deer. |
| 85 | êNbath-anjii | Eighty-five. Example: arubadh (60) miga + irovath-anjii (25) miga = 85 miga (deer) |
| 86 | êNbath-aar | Eighty-six. Example: êLubadh (70) thenge + padhinaar (16) thenge = 86 thenge (coconuts) |
| 87 | êNbath-êêL | Eighty-seven. Example: êLubadh (70) chettu + padhinêêL (17) chettu = 87 chettaachi (trees) |
| Peas | bataaNi | Peas |
| 88 | êNbath-êtt | Eighty-eight. Example: pathth (10) bataaNi + êLubath-êtt (78) bataaNi = 88 bataaNi (peas) |
| 89 | êNbath-woyimbadh | Eighty-nine. Example: irovadh (20) podiya + arubath-woyimbadh (69) podiya = 89 podiya (sarees) |
| 90 | Tombadhu | Ninety. Example: naapadh (40) patte + aimbadh (50) patte = 90 patte aachi (stripes) |

## Kodava Wedding (mangaLa)

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| A Kodava wedding is a two-day celebration. | Kodava mangaLa dhand dhina'thra samaarambha. | dhand: two, dhina: days. |
| Wedding eve ceremony | Uur kuuduva | uuru: village, kuuduva: to gather. |
| Main wedding ceremony | Muhuurtha | The second and main day of the wedding. |
| Mehndi / Henna | thaaraNa | The paste used for body art. |
| Mehndi ceremony | Thaarana MangaLa | A special ceremony in some parts of Kodagu. |
| Chopping banana stumps | BaaLe keththo | A ceremony on the muhuurtha day to honor the families. |
| Example sentence | Muhuurtha'thra polaaka baaLe keththu paddhati maadith, mangaLa-kaara pinynya mangaLa-karthi'ra wokka'k baaLe mariyaadhi koDupa. | On the muhuurtha morning, the baale keththu ritual is performed, and families honor each other. |
| The wedding happens on the muhuurtha day. | Muhuurtha'thl mangaLa-kaara pinynya mangaLa-karthi'ra mangaLa aapa. | Stating when the main ceremony occurs. |
| Water worship ritual | Ganga puuje | Ritual where the bride worships River Kaveri (Dakshina Ganga). |
| Traditional Kodava music | Vaalaga | The music played at traditional events. |
| Dance | AaTu | The Kodava dance, or any dance. |
| Happy / Enjoying | gauju | A state of happiness or enjoyment. |
| To enjoy / be happy | gauju maaduva | The act of enjoying oneself. |
| Relatives and family dance and enjoy at the wedding. | Bendhu pinynya samsaara aadith mangaLa'thl gauju maaduva. | Bendhu: relatives, samsaara: family, aadith: (they) dance. |

## Time Expressions

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| How to tell time? | Gante ennane ênnuva? | Question from instructor. |
| Morning | Polaaka | Morning |
| Midday | Nadu majjaNa | Noon |
| Afternoon | MajjaNa | Afternoon |
| Night | Baittu | Night |
| Midnight | Nadu paadhare | Midnight |
| What time is it now? | Ikka echchak gante? | Common question. |
| Now it's eight o'clock. | Ikka eTTu gante. | Answer example. |
| It's eight in the morning. | Ikka polaaka'thra eTTu gante. | Specific time of day. |
| Quarter (1/4) | Kaal | As in quarter past the hour. |
| Half (1/2) | Are | As in half past the hour. |
| Three quarters (3/4) | Mukkaal | As in quarter to the hour. |
| 8:15 | eTTe kaal | Eight and a quarter. |
| It's 7:30 now. | Ikka êLare gante. | êL (7) + are (half). |
| 3:45 | Idh muunde mukkaal. | It is three and three-quarters. |
| 11:15 | Pannande kaal | Quarter past eleven. |
