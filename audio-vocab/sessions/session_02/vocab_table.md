## Greetings & Phrases

| English | Kodava Takk | Explanation |
|---------|-------------|-------------|
| How are you? (plural / formal) | ninga ennane uLLira? | ninga = you (plural/formal), ennane = how, uLLira = are |
| How are you? (singular / informal) | niinu ennane uLLiya? | niinu = you (singular), ennane = how, uLLiya = are |
| We are fine. | nanga chaayitthe uNd | nanga = we, chaayitthe = fine/well, uNd = is/are |
| I am fine. | naanu chaayitthe uLLo | naanu = I, chaayitthe = fine/well, uLLo = am |
| I am fine. (alternative) | chaayillu uLLo | chaayillu = in wellness, uLLo = am |
| What is your name? | niida pedha enth? | niida = your, pedha = name, enth = what |
| What do you call me? / What's my name? | naada pedha enth'end nenne noaTana? | naada = my, pedha = name, enth'end = saying what, nenne = me, noaTana = call/address |
| My name is Smitha. | naada pedha Smitha | naada = my, pedha = name |
| What's up? / What news? | enne visheasha? | enne = what, visheasha = news/special |
| This | idhu | proximal demonstrative |
| That | adhu | remote demonstrative |
| This is an orange, that is a banana. | idhu puLi, adhu baaLe | idhu = this, puLi = orange, adhu = that, baaLe = banana |
| Which is the banana? | ikkaa eadhu baaLe? | ikkaa = now/here, eadhu = which, baaLe = banana |
| This is the banana. | idhu baaLe | idhu = this, baaLe = banana |
| What is that? | adhi enth'adhu? | adhi = that, enth'adhu = what is it |
| Grandfather's/grandmother's estate | thaathathaayi thaaTTa | thaatha = grandfather, thaayithaayi = grandmother's, thaaTTa = estate/plantation |
| Fruits from the estate | thaathathaayi thaaTTathra paNNu | thaathathaayi thaaTTathra = from the estate (genitive), paNNu = fruit(s) |
| What is there for me? / What do we have? | nange enth'onno? | nange = for us/me, enth'onno = what is there |
| Cooking | aDige | |
